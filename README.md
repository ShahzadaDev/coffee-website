# Mercado Coffee Bar — Website

A single-page website for **Mercado Coffee Bar**, located in Safa Mall Food Court, Malir Cantonment, Karachi.

## Features

- **Single-page layout** with smooth-scroll navigation
- **Full menu** with every drink and price:
  - Signature Hot Coffee
  - Iced Coffee Creations
  - Artisan Frappes
  - Matcha Collection
  - V60 Hand-Brew Experience
- **Clear calls-to-action** (Call to Order / Get Directions) including a sticky mobile CTA
- **Guest reviews** (5.0 rating)
- **Visit section** with address, phone, hours, and an embedded Google Map
- Fully **responsive** design with a warm coffee-house theme

## Tech

Plain HTML, CSS, and a small JavaScript file — no build step required.

| File | Purpose |
|------|---------|
| `index.html` | Page markup and content |
| `styles.css` | Styling and responsive layout |
| `script.js` | Mobile nav + dynamic footer year |

## Running locally

Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
